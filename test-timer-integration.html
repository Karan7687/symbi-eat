🎯 Order Timer Integration Test

Test Instructions:

  1. Go to http://localhost:3100
  2. Click the "🧪 Test Order Timer" button on the home page
  3. You should see a green timer bar appear in the navbar
  4. The timer should show a countdown from 08:00
  5. Click "Details" to expand more information
  6. Click the ❌ button to close the timer

✅ Expected Behavior:

  • Green timer bar slides down from top of page
  • Shows order number (ORD######)
  • Shows queue position (1)
  • Shows live countdown timer
  • Shows order status (Preparing)
  • Progress bar shows 75% completion
  • Details panel expands/collapses
  • Timer persists across page navigation

🔧 How it Works:

  • When you click "Test Order Timer", it triggers a custom event
  • The layout.ejs JavaScript listens for this event
  • Timer appears with simulated order data
  • Order data is stored in sessionStorage for persistence
  • Countdown updates every second
  • Timer stays visible until manually closed or order completes

🚀 Real-World Integration:

In the actual app, the timer will automatically appear when:

  • User places an actual order through the cart
  • Order is confirmed and added to the queue
  • Real order data comes from the database
  • Socket.IO provides real-time updates
  • Timer shows actual preparation time and queue position
